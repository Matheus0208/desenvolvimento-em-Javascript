# Desenvolvimento em JavaScript 💛

Este repositório é um espaço dedicado à prática, estudo e consolidação dos meus conhecimentos em **JavaScript**. Aqui, busco documentar minha evolução na linguagem, desde os fundamentos da sintaxe até conceitos avançados como manipulação de DOM e assincronismo.

---

## 🚀 Trilhas e Conceitos Abordados

Para organizar os estudos, os códigos e mini-projetos deste repositório estão divididos pelos seguintes conceitos principais. Use as caixas de seleção para acompanhar o progresso:

- [ ] **Fundamentos da Linguagem:**
  - Variáveis (`var`, `let`, `const`) e Tipos de Dados.
  - Operadores (Aritméticos, Lógicos e de Comparação).
  - Estruturas de Controle (`if/else`, `switch`) e Repetição (`for`, `while`).

- [ ] **Funções e Escopo:**
  - Declaração de funções clássicas e *Arrow Functions*.
  - Parâmetros, retornos e escopo de bloco/função.
  - *Callbacks* e funções de alta ordem.

- [ ] **Estruturas de Dados:**
  - Manipulação de Arrays (métodos como `map`, `filter`, `reduce`, `forEach`).
  - Objetos Literais e iteração sobre propriedades.

- [ ] **JavaScript Moderno (ES6+):**
  - Desestruturação (*Destructuring*).
  - Operadores *Spread* e *Rest*.
  - *Template Literals*.

- [ ] **Manipulação de DOM e Eventos (Front-end):**
  - Seleção de elementos (`querySelector`, `getElementById`).
  - Criação e remoção de nós no HTML.
  - Escuta de eventos (`addEventListener`).

- [ ] **Assincronismo:**
  - *Promises*.
  - Consumo de APIs com `Fetch API`.
  - Sintaxe `async` e `await`.
